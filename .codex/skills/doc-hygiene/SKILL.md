# Skill: Documentation Hygiene

## Goal
Keep active docs concise, current, and optimized for LLM consumption.

## Steps
1. Identify stale/superseded details in active docs.
2. Trim non-essential historical context from active files.
3. Move historical detail to archive docs when needed.
4. Normalize formatting and section ordering for readability.
5. Confirm cross-file consistency (`AGENTS.md`, `ROLES.md`, `ISSUES.md`, `docs/README.md`).

## Constraints
- Prioritize active operational context over narrative history.
- Avoid duplicating the same guidance across multiple active docs.
- Keep instructions explicit and machine-actionable.
